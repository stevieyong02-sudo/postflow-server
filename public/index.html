Run PostFlow MCP Server
Start your local PostFlow MCP server in terminal:
npm install @modelcontextprotocol/sdk zod express cors
node postflow-mcp-server.js
Add to Claude Desktop config:
{"mcpServers": {"postflow": {"command": "node", "args": ["postflow-mcp-server.js"]}}}
