Setup guide
Step 1 — Create Meta App (5 min)
Go to developers.facebook.com → Create App → Business type
Copy your App ID and App Secret

Step 2 — Set redirect URI
In your Meta App → Facebook Login → Settings → Valid OAuth Redirect URIs:
http://localhost:3333/auth/facebook/callback

Step 3 — Set env vars & start server
FB_APP_ID=your_app_id
FB_APP_SECRET=your_app_secret
BASE_URL=https://your-railway-url.up.railway.app
node server.js
